# U & I Salon

Beauty salon website with contact form and admin dashboard.

## Local Development

```bash
# Terminal 1 - Backend
cd backend && npm install && npm start

# Terminal 2 - Frontend
npm install && npm start
```

Admin: http://localhost:3000/admin/messages (password in .env)
